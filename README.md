# Chess GraphQL

## Getting started
```bash
# clones this repo
git clone https://github.com/ncksllvn/chess-graphql.git

# switch to new directory
cd chess-graphql/

# Make sure you're using the right version of Node (via Node Version Manager)
nvm use

# Set up your environment
cp sample.env .env

# start the site
npm start # -> localhost:4000
```
